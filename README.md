# The Books Library UI

A fully functional mock library management web app implemented with vanilla HTML/CSS/JS.

## Run

Open `index.html` in a browser.

## Included features

- Discover, category, my library, reserved, favorites, settings, help, admin pages.
- Borrow/reserve actions require authentication.
- Email/password and mock Google sign-in.
- Student verification with 3-attempt flow and profile merge.
- Admin dashboard with charts, user management, book CRUD, and activity logs.
- Reservation countdown (2-hour expiry).
